-- 核心依赖
flask：Web 框架，用于构建后端服务

flask-cors：处理跨域请求，解决前端与后端的跨域问题

requests：用于调用外部 API（如 DeepSeek 的聊天接口）

python-dotenv（可选）：用于管理环境变量（如 API 密钥），提升代码安全性

安装命令

可以通过以下命令安装所有依赖：

bash

pip install flask flask-cors requests python-dotenv

-- 补充说明

代码中未使用数据库、ORM 或其他复杂依赖，因此上述包已覆盖所有功能需求。

若需要部署到生产环境，建议额外安装gunicorn（WSGI 服务器）和eventlet（异步支持）：

bash

pip install gunicorn eventlet
